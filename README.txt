Each folder contains the 700 instances of its respective class 
marked with the name "BPP.v_c_x". Where v represents the class, 
c the capacity, and x the instance number (x = 1, ..., 100).

Files with name "BPP.v_c_x_opt" contain the optimal 
value of each of the 7 sets of instances.

Files with name "BPP.v_c" contain the names of the instances per set.

Files named "instances" within each folder contain the names of
the 700 instances of the class.
